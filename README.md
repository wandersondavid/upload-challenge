
# CarouselUploader

>  React + TypeScript + Vite + Storybook + Cypress

## Required global dependencies

- Node.js v18
- Npm

## Up & Running

1. Install local dependencies:

```terminal
npm install
```

2. Start:

```terminal
npm run dev
```

3. Test of componetens:

```terminal
npm run cypress:open
```

4. Build:

```terminal
npm run build
```

5. Start storybook:

```
npm run storybook
``````
